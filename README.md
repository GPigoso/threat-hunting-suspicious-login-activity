# Threat Hunting Lab – Suspicious Login Activity

## Overview

This project demonstrates a threat hunting investigation simulating a SOC Analyst proactively identifying suspicious login activity using authentication logs.

The objective was to detect potential brute force attacks and unauthorized access attempts.

## Tools Used

• SIEM concepts  
• Log analysis  
• CSV log investigation  
• Security incident documentation  

## Scenario

During proactive threat hunting, abnormal login patterns were identified involving multiple failed login attempts from an external IP address.

## Files included

• auth-log-sample.csv – authentication logs analyzed  
• investigation-notes.md – investigation process  
• threat-hunting-report.pdf – final incident report
